## Installation
Prerequisites:
*  Docker
*  Docker Compose

```
docker-compose up -d
```

## Build/Package
```
mvn clean package
```

## Testing
```
mvn test
```

## Used Dependencies
* Spring Boot Web
* Spring Boot Actuator
* Spring Boot Validation
* Spring Boot Data Couchbase
* Spring Boot Security
* Auth0 JWT
* Springdoc OpenApi (openapi-ui + security)
* Mapstruct
* Lombok

## Abilities
* JWT
* Couchbase Cache
